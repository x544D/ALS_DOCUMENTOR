-Len(wildcard)
[
Returns Length of an Enumerale Object!
if wildcard specified  the len will returns -1 if it does not respect that wildcard condition.
else it will return the original length of that Enumerable object .

wildcard can be :
* >x
* <x
* =x
* >=x
* <=x
* !x

where x is a number .

]
